# Scout Plugin

This repository is set up as a Claude Code plugin.

## Installation

Install via the plugin marketplace (recommended):

```bash
/plugin marketplace add Cemberk/scout
/plugin install scout@cemberk-scout
```

This will automatically install the Scout skill globally for use in all your projects.

## Manual Installation

If you prefer manual installation, see [plugins/scout/README.md](../plugins/scout/README.md) for detailed setup instructions.
