# Onboarding checklist

The onboarding checklist covers a new hire's first two weeks at ACME. Managers own the items assigned to them, and new hires own the items in the individual task list.

## Week 1

### Day 1
New hire tasks:
- Accept laptop
- Activate SSO
- Set up [[1password]]

Manager tasks:
- Post an intro message in `#new-hires`

### Day 2
New hire tasks:
- Read the [[employee-handbook]]
- Complete [[security-training]]

Manager tasks:
- Schedule 1:1s for weeks 2–4

### Day 3
New hire tasks:
- Read the team's runbooks
- Set up the local development environment

Manager tasks:
- Assign the first ramp-up task

### Day 5
New hire tasks:
- Submit a first pull request; a documentation fix is acceptable

Manager tasks:
- Review the pull request the same day

## Week 2

- Pair with a teammate on a real feature
- Attend the all-hands
- Attend the Friday wiki lint review
- Complete the context-engineering primer reading
- Meet the [[onboarding-buddy]] for lunch
