# ACME Onboarding Checklist

This checklist covers a new hire's first two weeks. Managers own items
in the "Manager" column; new hires own items in the "You" column.

## Week 1

| Day | You | Manager |
|---|---|---|
| Day 1 | Accept laptop, activate SSO, set up 1Password | Intro message in #new-hires |
| Day 2 | Read the Employee Handbook, complete security training | Schedule 1:1s for weeks 2–4 |
| Day 3 | Read the team's runbooks, set up local dev env | Assign first ramp-up task |
| Day 5 | Submit first PR (docs fix is fine) | Review PR same-day |

## Week 2

- Pair with a teammate on a real feature
- Attend the all-hands and the Friday wiki lint review
- Complete the context-engineering primer read
- Meet your onboarding buddy for lunch
