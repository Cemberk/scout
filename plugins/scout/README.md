# Scout Plugin for Claude Code

Company intelligence skill that navigates organizational knowledge sources and builds institutional memory.

## Installation

```bash
/plugin marketplace add Cemberk/scout
/plugin install scout@cemberk-scout
```

## Usage

Invoke with:
- `/scout find similar issues to JIRA-1234`
- `/scout who works on authentication?`
- `/scout save this error pattern`
- `/scout build knowledge base from this codebase`

See [SKILL_SETUP.md](SKILL_SETUP.md) for detailed setup and usage instructions.
